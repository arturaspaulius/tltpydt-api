# PYDT Serverless API

## Dev Setup / Deployment

* `npm install`
* `npm install -g serverless@1.70.0`
* Copy config.yml.template to config.yml and set the appropriate values.
* `npm run debug`

## Prod Deployment

* `npm run deploy-prod`
